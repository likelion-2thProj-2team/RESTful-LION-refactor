# RESTful Lion
> RESTful LION은 호텔 관리/예약 프로그램입니다.  
호텔 매니저는 간편한 인터페이스를 통해 호텔을 등록, 수정 및 삭제할 수 있습니다.  
일반 유저들은 다양한 호텔 옵션을 검색하고 필요에 따라 예약할 수 있습니다.
